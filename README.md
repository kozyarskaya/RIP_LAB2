# Лабораторная работа №2: Реактивное программирование и веб-клиенты

## Состав проекта
* **Java 21** 
* **Spring Boot 3.5.9**
*   **quote-server** (Сервис B): REST-сервис на порту 8081. Возвращает случайные цитаты.
*   **quote-client** (Сервис А): Клиентское приложение на порту 8080. Обращается к серверу через WebClient.

## Запуск
1.  Запустите `QuoteServerApplication` (порт 8081).
2.  Запустите `QuoteClientApplication` (порт 8080).
3.  Откройте POSTMAN и выполните GET запрос: `http://localhost:8080/api/client/quote`

## Особенности неоптимального кода (в Сервисе B)

**Класс:** `QuoteService.java`

1. **Чтение файла при каждом запросе**: `Files.readAllLines(filePath)` 
   - Читает ВЕСЬ файл `quotes.txt` с диска при КАЖДОМ запросе
   - Вместо кэширования при запуске приложения

2. **Использование Collections.shuffle()**: `Collections.shuffle(lines)`
   - Перемешивает ВСЮ коллекцию из N элементов за O(n) времени
   - Для выбора одного случайного элемента достаточно O(1)
